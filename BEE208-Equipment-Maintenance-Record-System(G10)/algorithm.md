# Algorithm for Equipment Maintenance Record System

## Algorithm

1. Start.

2. Display the title **Equipment Maintenance Record System**.

3. Ask the user to enter the number of maintenance records.

4. Repeat the following steps for each maintenance record:

   * Enter the equipment name.
   * Enter the equipment ID.
   * Enter the fault description.
   * Enter the technician name.
   * Enter the maintenance cost.
   * If the maintenance cost is less than 0, display an error message and ask the user to enter the cost again.
   * Enter the repair status.
   * If the repair status is not **Pending**, **In Progress**, or **Completed**, display **Invalid Status**.

5. Display the maintenance report for all records entered.

6. Save all maintenance records into a file named **maintenance_records.txt**.

7. Display a message that the records have been saved successfully.

8. Stop.
