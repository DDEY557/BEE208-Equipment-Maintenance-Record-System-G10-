# EQUIPMENT MAINTENANCE RECORD SYSTEM

## 1. Introduction
This project is a C++ application developed to manage equipment maintenance records in laboratories, workshops, and engineering environments. It helps users store, display, and save maintenance information in a structured way.



## 2. Problem Statement
Manual recording of equipment maintenance is inefficient and prone to errors. Records can be lost or difficult to retrieve. This system solves that problem by digitizing the process.



## 3. Aim of the Project
To develop a C++ program that records, displays, and stores maintenance information for electrical and electronic equipment.



## 4. Objectives
- Capture maintenance records
- Validate input data
- Display maintenance reports
- Save records to a file
- Use OOP concepts in C++



## 5. System Requirements
### Functional Requirements:
- Input equipment details
- Validate maintenance cost
- Validate repair status
- Save data to file

### Non-Functional Requirements:
- Simple user interface
- Fast execution
- Reliable data storage
